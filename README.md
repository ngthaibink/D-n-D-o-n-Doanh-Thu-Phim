Dự đoán doanh thu phim
Đây là đồ án cuối kì môn lập trình Python . Dùng phân tích dữ liệu từ các bộ phim đã chiếu để dự đoán ra doanh thu phim dựa trên các chỉ số votes, điểm,ngân sách,..
Gồm bốn mục:
-Làm sạch và phân loại dữ liệu
- EDA , vẽ các biểu đồ để thấy sự tương quan
- Tạo model dự đoán, tìm ra model tốt nhất
- Xây dựng website để điền các thông tin cần dự đoán
